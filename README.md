📌 Descripción

BecoMarket es una aplicación de gestión comercial desarrollada en Java con una base de datos MySQL administrada mediante XAMPP. El proyecto está diseñado para facilitar la administración de un negocio, permitiendo gestionar usuarios, productos y pedidos de forma eficiente.

Su objetivo principal es optimizar las operaciones diarias de un comercio mediante una interfaz intuitiva y herramientas que simplifican el control del inventario y el seguimiento de las ventas.

🚀 Tecnologías utilizadas
Java - Desarrollo de la aplicación.
MySQL - Sistema de gestión de bases de datos.
XAMPP - Entorno de desarrollo para la administración de MySQL.
JDBC - Conexión entre Java y la base de datos.
Funcionalidades principales
- Registro e inicio de sesión de usuarios.
- Gestión de productos e inventario.
- Creación y seguimiento de pedidos.
- Consulta de información de productos.
- Administración de datos comerciales.
- Persistencia de información mediante MySQL.
